These files can be used to do the first simulation. 
1. The files in the folder "generatData" can be used to generate data.

2. The R file "simuFunc_minorR.R" can be used to run the simulation analysis. Each simulation run for a specific set of parameter values may take 10-20 hours.

3. The R file "summary_minorR.R" can be used to summarize the simulation results.
