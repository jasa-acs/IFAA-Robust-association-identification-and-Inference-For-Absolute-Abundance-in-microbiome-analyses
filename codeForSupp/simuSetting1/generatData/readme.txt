These R files can be used to generate data for the first simulation. 
1. implement the file "dataGenerat_minorR.R" with "generateDataAncom(batch=1)" will generate 100 data sets stored in the file "IFAAancom1.csv".

2. The sequencing depth can be adjusted in the function "getSeqDep()" in the file "dataGeneratUtils_minorR.R".

A new set of parameter values can be generated by running the file "betaGenerat_minorR.R".
