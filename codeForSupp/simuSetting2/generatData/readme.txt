These files can be used to generate the data and parameters values for the simulation by the following these steps:
1. run the file "dataGenerate_revision_minor.R" with "batch=1" and "startFromScratch=T". This will generate 50 data sets in batch 1 using the parameter values presented in the paper. These data will be stored in the file "IFAA_Rev_origin1.csv"
2. run the file "dataGenerate_revision_minor.R" with "batch=1" and "startFromScratch=F". This will generate 50 data sets in batch 1 using the parameter values presented in the paper. The sparsity level will be about 47%. These data will be stored in the file "IFAA_Rev1.csv".
3. In the conBern() function of the file "detaGeneratUtils_Revison_minor.R", comment out "cutVec=rep(qnorm(0.44),nTaxa)" and uncomment "cutVec=rep(qnorm(0.64),nTaxa)". Then run the file "dataGenerate_revision_minor.R" with "batch=2" and "startFromScratch=F". This will generate 50 data sets in batch 2 using the parameter values presented in the paper. The sparsity level will be about 66%. These data will be stored in the file "IFAA_Rev2.csv".

Different sparsities can be controlled in the corBern() function. A new set of true parameters values can be generated by running the file "betaGenerat_Revision_minor.R". Sample size and parameter values can be changed in the file "TurParam_Revision_minor.R".
