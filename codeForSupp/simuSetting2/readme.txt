The files in this folder can be used to run and summary the second simulation stduy.

1. Data can be generated with the files in the subfolder "generatData".

2. R code for running the simulation analysis is in the file "simu_Rev_minor.R" Each simulation run for a specific set of parameter values may take 20-40 hours.

3. R code for summarizing the simulation results is in the file "summary_Rev_minor.R"
